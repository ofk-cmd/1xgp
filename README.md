# Лендинг-копия Google Play — раздача APK 1xBet (KG + TJ)

Страница имитирует листинг приложения в Google Play. Кнопка «Установить» ведёт
на трекинговую CRM-ссылку, **собранную динамически из UTM-меток в URL лендинга**.

## Файлы
- `build_playmarket.py` — генератор (правишь контент здесь, прогоняешь — пересобирает HTML).
- `1xbet_kg.html`, `1xbet_tj.html` — готовые страницы (CSS/JS внутри, ассеты — из `assets/`).
- `assets/` — `icon.png` (144px) + `screen1..6.webp` (640px) — официальные, оптимизированы под веб (~330 КБ).
- Пересборка: `python3 build_playmarket.py`
- Подробное описание для ревью — `../mobile assets/Лендинг — описание для ревью.md`.

## Как работает кнопка (механика из доки)
Лендинг получает на вход:
```
https://1xbt.kz/?domain=reffpa.com/L&tag=...&pb=...&click_id={clickid}
```
и кнопка выдаёт ровно CRM-ссылку:
```
https://reffpa.com/L?tag=...&pb=...&click_id={clickid}
```
- `domain` — декодируется один раз (склейка `https://{domain}?...`).
- `tag` (со скобками `[]`), `pb`, `click_id` (с макросом `{clickid}`) — передаются **СЫРЫМИ**, без перекодирования. Любые лишние параметры тоже прокидываются как есть.
- `geo` — служебный, в ссылку не попадает.
- Если UTM-меток нет → кнопка ведёт на `FALLBACK` (по умолчанию `https://reffpa.com/L`; меняется в `build_playmarket.py` → `COMMON["fallback"]`).

Собираешь трекинговую ссылку: к домену лендинга (`https://<твой-домен>/`) добавляешь
`?domain=...&tag=...&pb=...&click_id={clickid}` — значения один-в-один из CRM-ссылки.

## Ассеты
Стоят **официальные** ассеты, оптимизированные под веб (`assets/`):
- `icon.png` — 144px, из `../mobile assets/Logo 1024x1024.png`
- `screen1..6.webp` — 640px WebP, из `../mobile assets/ru/6_9_screen_*.png`

Заменить — положить новые файлы в `assets/` под теми же именами. Оригиналы (1024px иконка,
скрины 3 размеров) — в `../mobile assets/`. Скриншоты, кроме первого, грузятся лениво.

## Контент по гео
Правится в `build_playmarket.py` → словарь `GEOS` (отзывы, downloads, бонус, offer_short) и `COMMON`
(имя, разработчик, рейтинг, размер, дата). Бонусы:
- **KG:** 120% до 35 000 KGS
- **TJ:** 120% до 4 000 TJS

## Деплой
На KG-домен кладёшь `1xbet_kg.html` (как `index.html`) **+ папку `assets/`**;
на TJ-домен — `1xbet_tj.html` + `assets/`. Трекинговую ссылку строишь от своего домена (см. выше).

## Заметки
- 1xBet **нет в Google Play** (гемблинг там забанен) — поэтому фейк-страница Play и есть
  стандартный способ раздачи APK. Дизайн скопирован с актуального листинга Play (M3).
- 18+ дисклеймер в подвале. Под TJ при необходимости можно добавить лиц-плашку TOJ TURON
  (в Play-странице обычно не показывается — оставлено чисто).
- `<meta robots noindex,nofollow>` стоит — страницу не индексируем.
